# SpecTrail v0.1.0

SpecTrail v0.1.0 is the first public baseline of the local-first, source-grounded ReqIR extraction and review workflow.

## Highlights

- Parse Markdown, DOCX, and text-based PDF documents into canonical text and `DocumentBlock` records.
- Extract ReqIR through deterministic mock, replayable recorded, or OpenAI-compatible live model modes.
- Validate schema, EARS structure, and exact source quotes before exporting results.
- Review, approve, reject, restore, or edit requirements in the local Review UI.
- Export traceable ReqIR JSON, source maps, review logs, and Excel workbooks.

### Real live run summary

```text
Input: docs/sample_srs.md
Mode: live
Model: DeepSeek-V4-Flash
Prompt: reqir_extraction_v1
Parser: markdown_parser_v1

Parsed blocks: 22
Raw requirements: 15
Validated requirements: 15
Source quote passed: 15/15
Source quote failed: 0
Validation errors: 0
Validation warnings: 5
```

All live-run confidence values were numeric. Four warnings recorded safe removal of Markdown list markers from source quotes, and one warning flagged an event-driven requirement without an explicit trigger. The source quote validator still produced `PASS_EXACT` for all 15 requirements.

The configured provider uses a self-signed certificate chain, so this local smoke run used the documented `--insecure` option. This option is not recommended for providers with a publicly trusted certificate chain.

## Release artifacts

- `reqir.json`: validated, unreviewed ReqIR snapshot generated from the public sample document.
- `requirements.xlsx`: 17-column traceability workbook generated from the same run.
- Source archives generated by GitHub from the `v0.1.0` tag.

## Known limitations

- Scanned PDFs and OCR are not supported.
- PDF page context is best-effort; bounding-box highlighting is not implemented.
- The current workflow processes one document per task.
- Live model output remains subject to human review before downstream use.
- Authentication, persistent database storage, async queues, ReqIF, SysML, and Gherkin export are not included.
